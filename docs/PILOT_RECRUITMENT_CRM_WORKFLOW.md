# Pilot Recruitment CRM Workflow

## Pipeline Stages
1. Contact Identified
2. Outreach Sent
3. Responded
4. Discovery Scheduled
5. Qualified
6. Onboarding Scheduled
7. Live Pilot

## Required Fields
- Contact name
- Church name
- Role/title
- Channel used
- Last touch date
- Next action date
- Pipeline stage

## Weekly Review Ritual
- Monday: new outreach batch + stale follow-ups
- Wednesday: pipeline review and call prep
- Friday: conversion review + adjustments

## KPIs
- Outreach sent count
- Response rate
- Discovery call count
- Qualification rate
- Onboarding conversion rate
