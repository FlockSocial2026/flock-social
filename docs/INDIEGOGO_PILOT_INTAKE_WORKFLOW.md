# Indiegogo Pilot Intake Workflow

## Step 1 — Intake
- Capture church/contact source (campaign comment, DM, referral, outreach).
- Record fit details (size, leadership responsiveness, pain points).

## Step 2 — Qualification
- Run discovery call using existing notes template.
- Score readiness: High / Medium / Low.

## Step 3 — Onboarding Queue
- If High/Medium, assign onboarding owner and kickoff target week.
- Send onboarding packet and required prep checklist.

## Step 4 — Activation
- Execute bootstrap + first communication/event setup.
- Capture baseline KPIs for week-1 comparison.

## Step 5 — Review
- Confirm first-week adoption indicators.
- Move to active pilot or remediation track.
