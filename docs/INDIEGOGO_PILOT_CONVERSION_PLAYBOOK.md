# Indiegogo Pilot Conversion Playbook

## Stage 1 — Identify
- Tag interested churches/supporters from comments, DMs, and outreach replies.
- Score by fit (size, engagement readiness, leadership responsiveness).

## Stage 2 — Qualify
- Run 20-30 minute discovery call.
- Confirm communication pain points and pilot readiness criteria.

## Stage 3 — Commit
- Send pilot onboarding packet.
- Confirm owner + start week.
- Record commitment status in tracker.

## Stage 4 — Activate
- Begin onboarding flow.
- Set first KPI baseline capture window.
- Schedule week-1 check-in.

## Stage 5 — Prove
- Capture initial outcomes.
- Convert outcomes into campaign proof update.
