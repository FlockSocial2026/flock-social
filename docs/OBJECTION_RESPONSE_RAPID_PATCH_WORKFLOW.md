# Objection-Response Rapid Patch Workflow

## Trigger
- Same objection appears 3+ times in a 6-12h window.

## Workflow
1. Capture objection wording and context.
2. Draft response update (FAQ + outreach snippet).
3. Approve and deploy patch to active messaging assets.
4. Log patch time and measure post-patch sentiment/conversion.

## SLA
- Detect to patch deploy: <= 2 hours.
