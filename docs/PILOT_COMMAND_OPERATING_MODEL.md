# Pilot Command Operating Model

## Mission
Drive measurable pilot outcomes across participating churches with disciplined weekly execution and rapid issue resolution.

## Operating Layers
### Layer 1 — Church Delivery
- Onboarding completion
- Weekly announcements/events cadence
- Member engagement signal tracking

### Layer 2 — Performance Management
- Church-scoped KPI review
- Trend deltas and exception detection
- Corrective action assignment

### Layer 3 — Governance
- Role/permission integrity checks
- Audit visibility reviews
- Escalation and blocker management

## Weekly Rhythm
- Monday: KPI review + plan
- Midweek: execution check + blocker clearance
- Friday: outcome review + next-week setup
