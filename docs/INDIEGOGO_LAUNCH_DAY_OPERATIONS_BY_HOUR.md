# Indiegogo Launch-Day Operations (By Hour)

## T-3h
- Final QA pass (copy, links, mobile/desktop render)
- Confirm all media loaded correctly
- Confirm day-1 outreach list ready

## T-2h
- Final Go/No-Go review using checklist
- Lock decision and timestamp
- Stage launch announcements with campaign URL

## T-1h
- Soft internal dry-run: open campaign page from multiple devices
- Confirm comment/reply workflow owner

## T0 (Launch)
- Publish campaign
- Send email announcement
- Post social announcement
- Send warm-network DM wave

## T+1h
- Verify campaign analytics and page rendering
- Respond to initial comments/questions
- Log first momentum metrics

## T+3h
- Send short follow-up post/update
- Process objections using response bank
- Track conversion from outreach channels

## T+6h
- Review performance snapshot
- Decide if CTA/copy micro-adjustments needed

## T+12h
- Evening follow-up announcement
- Refresh responder list and engage top prospects

## T+24h
- Publish first-day recap (brief)
- Compare performance against expected benchmarks
