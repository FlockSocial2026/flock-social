# Indiegogo Growth Loop Map

## Loop A — Acquisition -> Backing
1. Publish outreach/update content
2. Drive traffic from top channels
3. Convert with V3 page + proof blocks
4. Capture objections and patch messaging
5. Recycle improved messaging into next outreach wave

## Loop B — Trust -> Conversion
1. Share weekly transparent KPI updates
2. Demonstrate shipped execution milestones
3. Increase backer confidence and shares
4. Gain new warm introductions
5. Feed introductions into outreach pipeline

## Loop C — Campaign -> Pilot Pipeline
1. Identify churches/high-intent supporters
2. Run qualification call
3. Move qualified orgs to onboarding queue
4. Publish pilot progress signals
5. Reinforce campaign credibility

## Loop KPIs
- Traffic by channel
- Conversion rate by channel
- Backer-to-intro referral rate
- Qualified pilot conversations
