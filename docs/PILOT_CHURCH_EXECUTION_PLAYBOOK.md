# Pilot Church Execution Playbook

## Cohort Design
- Target: 3-5 local churches
- Segment mix: small, medium, and growth-stage congregations
- Duration: 6-8 weeks

## Onboarding Sequence
1. Church bootstrap + admin role assignment
2. Staff setup (pastor/staff/group leaders)
3. Weekly announcement cadence setup
4. Event calendar seed for next 30 days
5. Member invite campaign

## Weekly Operating Rhythm
- Monday: Weekly push announcement
- Wednesday: Midweek reminder + prayer prompt
- Friday: Weekend event/service reminders
- Sunday: Follow-up + next-week teaser

## Data Capture
- Weekly metrics pull from `/api/metrics/pilot-summary`
- Qualitative feedback capture from church staff
- Incident log for product friction points

## Exit Package Per Pilot
- Before/after engagement snapshot
- Testimonial quote(s)
- Identified wins + blockers
- Rollout recommendation
