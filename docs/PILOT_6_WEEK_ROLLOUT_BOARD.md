# Pilot 6-Week Rollout Board

## Week 0 (Pre-Launch)
- [ ] Finalize church shortlist (3-5)
- [ ] Send outreach and schedule intro calls
- [ ] Assign internal owners per church
- [ ] Verify onboarding packet and KPI templates

## Week 1 (Activation)
- [ ] Church bootstrap complete
- [ ] Admin roles assigned
- [ ] First weekly push published
- [ ] First event created

## Week 2 (Adoption)
- [ ] Member invite push
- [ ] Prayer/group engagement prompt
- [ ] Check role audit and issue log

## Week 3 (Stabilize)
- [ ] KPI review with church leadership
- [ ] Address workflow friction points
- [ ] Tune communication cadence

## Week 4 (Expand)
- [ ] Introduce second engagement campaign
- [ ] Capture qualitative testimony
- [ ] Compare trend deltas vs Week 1 baseline

## Week 5 (Optimize)
- [ ] Increase RSVP conversion with reminder timing tests
- [ ] Improve post/announcement quality cadence
- [ ] Prepare case-study draft

## Week 6 (Closeout)
- [ ] Final KPI snapshot + trend analysis
- [ ] Leadership debrief
- [ ] Document rollout recommendation (continue/expand/adjust)
