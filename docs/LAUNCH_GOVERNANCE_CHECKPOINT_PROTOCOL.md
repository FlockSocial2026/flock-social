# Launch Governance Checkpoint Protocol

## At each checkpoint
1. Update KPI watchboard.
2. Review open incidents and escalation states.
3. Confirm SLA compliance for inbound responses.
4. Decide keep/adjust/revert for active experiments.
5. Log decision and owner.

## Escalation Conditions
- Conversion trend turns red
- Sev1 incident open >1 hour
- Response backlog exceeds SLA

## Governance Output
- Checkpoint summary note
- Action list with owners and due times
