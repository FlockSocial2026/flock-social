# Indiegogo First 48 Hours Monitoring Protocol

## Monitoring Cadence
- Hour 0-6: check every 60 minutes
- Hour 6-24: check every 2-3 hours
- Hour 24-48: check every 4-6 hours

## What to Monitor
1. Funding progress velocity
2. Traffic source performance
3. Conversion by channel (email/social/DM)
4. Frequently repeated objections/questions
5. Page friction points (drop-off indicators)

## Response Rules
- Reply to comments/questions within same day
- If a repeated objection appears 3+ times, update FAQ and announcement copy
- If a channel underperforms materially, reallocate effort to stronger channel within 12h

## 24h Review Output
- Performance summary
- Top-performing channel
- Top objection theme
- Planned optimization actions

## 48h Review Output
- Funding velocity trend
- Channel mix recommendations
- Copy/CTA adjustments for week 1
- Updated outreach priority list
