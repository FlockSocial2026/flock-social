# Problem/Solution Evidence Sheet

## Problem 1: Fragmented Church Communications
Evidence:
- Churches juggle announcements/events across separate channels.
Solution:
- Flock announcements + events APIs and admin composer shipped.

## Problem 2: Inconsistent Member Engagement
Evidence:
- Low weekly cadence and weak follow-up loops.
Solution:
- Pilot ops cadence docs, outreach/playbook, and weekly KPI templates shipped.

## Problem 3: Lack of Leadership Controls
Evidence:
- Generic social tools lack church role governance.
Solution:
- Role model (member/group_leader/pastor_staff/church_admin), role assignment APIs, audit trail.

## Problem 4: Hard to Prove Outcomes
Evidence:
- Many products offer claims without measurable reporting.
Solution:
- `GET /api/metrics/pilot-summary` + CSV exports + trend deltas + admin metrics panel.
