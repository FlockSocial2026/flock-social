# Pilot Church Weekly Execution Cadence

## Monday — Planning & Setup
- Confirm weekly announcement objectives
- Confirm event pipeline for the week
- Set KPI targets per church

## Tuesday/Wednesday — Execution Check
- Verify announcements/events published on schedule
- Validate role permissions and owner coverage
- Capture early engagement signals

## Thursday — Intervention Window
- Identify underperforming churches/workflows
- Deploy corrective actions (messaging/process/support)

## Friday — Review & Rollforward
- Record KPI outcomes
- Log blockers and resolutions
- Prepare next-week action plan
