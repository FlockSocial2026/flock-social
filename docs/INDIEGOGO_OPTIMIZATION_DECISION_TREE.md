# Indiegogo Optimization Decision Tree

## Step 1 — Diagnose
- Is traffic low?
- Is conversion low?
- Is objection volume high?

## Step 2 — Choose Lever
### If traffic low
- Increase outbound volume to top-performing channel
- Reactivate warm network DMs

### If conversion low
- Switch hero/CTA variant
- Simplify ask in announcement copy
- Add fresh proof block near CTA

### If objections high
- Update FAQ immediately
- Add objection-response snippets to outreach copy
- Publish a trust/clarity mini-update

## Step 3 — Measure (next 12-24h)
- Funding velocity
- Channel conversion
- Response sentiment

## Step 4 — Decide
- Keep change if positive signal
- Revert and test alternate if neutral/negative
