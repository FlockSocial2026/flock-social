# Indiegogo Asset Production Order (Execution Sequence)

## Phase 1 — Core Copy Lock (Day 0)
1. Finalize `INDIEGOGO_MASTER_PAGE_V3.md`
2. Select hero option and primary CTA
3. Confirm perks + use-of-funds percentages

## Phase 2 — Visual Assets (Days 1-2)
1. 60-90s intro video script + shot list
2. Problem/solution graphic
3. Product screenshots (member + admin views)
4. Milestone timeline visual
5. Pilot-plan infographic

## Phase 3 — Page Build (Day 3)
1. Paste V3 copy into Indiegogo draft
2. Insert visuals in narrative order
3. Add perks tiers and FAQ
4. Run copy QA for clarity and consistency

## Phase 4 — Trust Layer (Day 4)
1. Add weekly update commitment block
2. Add use-of-funds visual
3. Add milestone proof references

## Phase 5 — Launch Readiness (Day 5)
1. Final proofread
2. Mobile/desktop page pass
3. Day-1 supporter outreach list finalized
4. Launch-day runbook reviewed
