# Indiegogo Go/No-Go Checklist

## Decision Rule
- **GO** only if all Critical items pass.
- **NO-GO** if any Critical item fails.
- **CONDITIONAL GO** allowed only for non-critical items with documented mitigation and owner.

## Critical (Must Pass)
- [ ] Master page copy finalized (`INDIEGOGO_MASTER_PAGE_V3.md`)
- [ ] Campaign URL and all links verified
- [ ] Video asset exported and uploaded
- [ ] Core screenshots/graphics uploaded and readable
- [ ] Perks and pricing finalized
- [ ] Use-of-funds finalized
- [ ] FAQ finalized
- [ ] Day-1 announcement pack finalized with live link
- [ ] Owner assignments complete for first 48h response window

## High Priority (Should Pass)
- [ ] Mobile and desktop QA pass complete
- [ ] Objection-response bank reviewed
- [ ] Week-1 update draft pre-staged
- [ ] Day-2 outreach follow-up prepared
