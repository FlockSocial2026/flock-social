# Pilot Follow-Up Cadence

## Day 0 (Outreach Sent)
- Send initial outreach message

## Day 2
- Follow-up #1 (value reminder + call CTA)

## Day 5
- Follow-up #2 (shorter, direct scheduling link)

## Day 9
- Final touchpoint (close loop respectfully)

## Post-Call Cadence
- Within 24h: send recap + next steps
- Within 72h: confirm onboarding timeline
- Weekly: send KPI summary once pilot starts
